# Pulse

![Pulse Logo](./logo.svg)

## Tagline

**See attention before everyone else.**

---

## Vision

Pulse is the attention intelligence platform for creators.

Most tools help creators write content.

Pulse helps creators discover opportunities before they become trends.

Instead of asking:

> "What should I write?"

Pulse answers:

> "What should I post right now to maximize attention?"

---

## Mission

Build the operating system for attention.

Pulse helps creators:

- Discover trends early
- Analyze viral content
- Generate winning ideas
- Improve hooks
- Understand audiences
- Track competitors
- Predict content performance

---

## Problem

Creators spend hours:

- Searching for ideas
- Studying viral posts
- Tracking competitors
- Guessing what will work
- Testing content blindly

The internet already tells us what works.

Most people just cannot see the patterns.

Pulse surfaces those patterns.
